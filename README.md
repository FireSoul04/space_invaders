# Space invaders

This a simple implementation of Space invaders using C++ and SDL2 as main library.
For now has plenty of bugs.

#### Requirements:
* Cmake 3.15 or newer
* SDL2 and SDL2-ttf

#### How to build:

Go to project root folder
```
mkdir build
cd ./build
cmake ../src
cmake --build .
```

To execute
```
./space_invaders
```
